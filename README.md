# AWS Intro Workshop Landing

Single-page static website prepared for an AWS Amplify deployment demo.

## Run locally
Open `index.html` in your browser.

## Main message
Welcome to the Introduction to AWS Workshop

## Deploy to AWS Amplify (quick path)
1. Push this repository to GitHub.
2. In AWS Amplify, choose **Host web app**.
3. Connect your GitHub repository and select the main branch.
4. Use the default static hosting settings.
5. Deploy and open the generated Amplify URL.
